# Tic Tac Toe Kata

## Kata description

### Game Setup

- One board – 3 x 3 cells
- Two players - O & X:
  - Player One - X
  - Player Two - O
- Players take turns

### Game Rules

- Player One starts
- A mark can only be placed on an empty cell
- Game ends if …
  - board full, or
  - only one symbol in column, row or diagonal - this
 is a winning condition: owner of the symbol wins
